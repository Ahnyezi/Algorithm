# 탐색 알고리즘

> 너비 우선 탐색
- 너비를 우선으로 하여 탐색을 수행
- '최단 경로'를 찾아주기 때문에, 최단 길이 보장에 사용
- queue로 구현

> 깊이 우선 탐색
- 보다 깊은 것을 우선적으로 탐색
- 너비 우선 : queue, 깊이 우선 : stack
- 하지만, stack 사용하지 않아도 재귀로 구현 가능
    - 컴퓨터는 구조적으로 항상 stack의 원리를 사용
    - 따라서, stack에 담고 빼는 것과 동일한 효과를 낸다.
